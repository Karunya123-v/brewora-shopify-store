## 📌 Project Overview

- Clean and modern UI/UX design
- Product listing with pricing and images
- Featured collections
- Responsive homepage layout
- Branding with consistent color palette

## 🛠️ Tools Used
- Shopify (Dawn Theme)
- Canva (Logo design)
- Unsplash (Product images)

## 🎯 Objective
To build a professional, ready-to-sell eCommerce store focusing on user experience and branding.
